### Requirements
 - A computer running Windows 10/11 or Linux (Wayland and X11 are supported)
 - Patience

### How to use
 - Download the app through github release page

### How to compile (Linux only, on Windows it's probably just as easy)
 - Install rust
 - `cargo build --release`
